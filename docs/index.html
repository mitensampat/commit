Commit v1.5.0

Commit finds the promises in your WhatsApp — yours and theirs. it notices when they're done, and surfaces the ones going quiet.

features

  1. finds commitments automatically — both directions. closes one on its own only when the chat shows both sides; anything less is a quiet suggestion
  2. "act on today" — opens with the 3-5 things worth acting on now, not a list of 600
  3. morning digest — one WhatsApp message a day. quiet days send nothing
  4. @find — ask anything about your chats, from WhatsApp or the dashboard
  5. @schedule <name> — find time with anyone, without ever leaving WhatsApp
  6. yours alone — messages stay on your machine, passcode-protected, API key encrypted
  7. menu bar app (mac) and system tray (windows) — see at a glance it's running

simple setup

scan a QR code, same as WhatsApp Web. Commit runs on your machine and reads new messages every 10 seconds.

the dashboard groups everything by person: what you owe, what they owe, how long it's been. reply, remind, mark done, dismiss.

scheduling, without the scheduling link new

someone asks to meet. you can send a booking link — which quietly says my time matters more than yours, go find a slot — or you message yourself: @schedule <name>.

Commit reads the thread, checks your calendar, and drafts three times you're free in your texting voice. say "propose" — it goes as an ordinary message from you. they reply "2". say "yes". booked.

they never see a link or a bot. just you, replying with times like someone who checked. two words, start to finish.

a note on privacy

messages are decrypted on your machine and stored in a local SQLite database. Commit uploads nothing.

message content goes to the Claude API for extraction — Anthropic does not train on API inputs.

requirements

macOS 12+ or Windows 10+. a WhatsApp account. a Claude API key. Google Calendar if you want @schedule.
