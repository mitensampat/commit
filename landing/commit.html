Commit v1.1.2

Commit enhances your WhatsApp usage by finding commitments automatically — things you said you'd do, and things others said they'd do. it auto detects when things are done.

when something goes quiet, it surfaces it for follow-up.

how it works

you scan a QR code to link your WhatsApp account, same as WhatsApp Web. Commit runs on your machine. every 10 seconds it reads new messages, sends them to the Claude API for analysis, and logs any commitments it finds.

the dashboard shows everything grouped by person: what you owe, what they owe, how long it's been. you can reply, set reminders, mark things done, or dismiss them.

features

  1. auto-extraction and auto-resolution of commitments
  2. follow-up nudges for things that have gone quiet
  3. reminders via WhatsApp message to yourself
  4. reply to any chat from the dashboard
  5. WhatsApp bot commands (message yourself: "commitments", "owe @name", "done <text>")
  6. favorites, search, dark mode, mobile web
  7. passcode protection, AES-GCM encrypted API key

a note on privacy

messages are decrypted on your machine and stored in a local SQLite database. nothing is uploaded to a server by Commit.

message content is sent to the Claude API for commitment extraction. Anthropic does not train on API inputs.

requirements

macOS 12+ or Windows 10+. a WhatsApp account. a Claude API key.
