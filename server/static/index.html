Welcome to Commit
Every conversation has promises buried in it. Commit finds them, tracks them, and closes them when they're done.
How it works

Reads your chats

Connects to WhatsApp and scans conversations for commitments.

Extracts automatically

Uses Claude to find who promised what, with the original quote.

Auto-resolves

When a commitment is fulfilled in the conversation, Commit marks it done.

Your data stays yours
Commit runs entirely on your machine. Messages are decrypted locally and stored in a local database. Message content is sent to the Claude API for commitment analysis — Anthropic does not train on API inputs. No cloud storage. No telemetry.
